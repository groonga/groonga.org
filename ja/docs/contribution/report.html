11.1. バグレポートの送り方¶

There are two ways to report a bug:

  • Submit a bug to the issue tracker
  • Report a bug to the mailing list

You can use either way It makes no difference to us.

11.1.1. Submit a bug to the issue tracker¶

Groonga project has two issue tracking systems, Redmine and GitHub issue tracker. Redmine is for Japanese and GitHub issue tracker is for English. You can use one of them to report a bug.

11.1.2. Report a bug to the mailing list¶

Groonga project has コミュニティ for discussing about groonga. Please send a mail that describes a bug.
